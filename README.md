## About The Project

<br/>
Spring blog is simple blog app to practice spring boot way to develop monolith
backend

The 2 principles of project:

- Follow spring boot 3 tier pattern as long as you can
- Follow react principles as long as you can

<!-- GETTING STARTED -->

## Getting Started

### Prerequisites

-   JDK 17 or upper
-   Maven

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/onuraltuntasb/springblog
    ```
2. Run 
 ```sh
     mvn spring-boot:run
   ```
